# YYYY-MM-DD Learning Journal

## 1. What I Did Today

Record what actually happened today. Distinguish input and output:

- **Input**: Watching videos, reading books, listening to explanations
- **Output**: Doing problems, writing code, reviewing, organizing notes

## 2. Where I Got Stuck

Record knowledge blockers, execution blockers, and state issues.

You can write:
- Which knowledge point you don't understand
- Which problem you can't solve
- Which code you can't write
- Whether phone/B站/抖音 distracted you
- Whether schedule or mental state affected learning
- Whether task was too vague, didn't know where to start

## 3. AI Judgment

The leader AI judges today's biggest problem based on the report.

Common types:
- Task ambiguity
- Over-planning
- Phone distraction
- Schedule impact
- Knowledge blocker
- Avoidance
- Output insufficient

AI can directly point out problems, but only criticize behavior and strategy, not personality.

## 4. Tomorrow's Plan

Only write two types of tasks:

- **Minimum task**: Small enough to start, must try to complete
- **Advanced task**: Do when state is good

If there are due reviews, prioritize them in minimum tasks.
