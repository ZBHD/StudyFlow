# Mistake Index

Last updated: YYYY-MM-DD

## Usage

This file indexes all mistakes. Detailed analysis goes in subject-specific files.

## Index Format

| Date | Subject | Mistake | Type | Status | Details |
|------|---------|---------|------|--------|---------|
| YYYY-MM-DD | [Subject] | [Mistake description] | [Type] | [Status] | [Link] |

## Status Description

- **To review**: Discovered but not yet analyzed
- **To practice**: Analyzed, entered review queue
- **Partial**: Reviewed but still unstable
- **Mastered**: No longer priority for review
- **Upgraded to weak point**: Appeared 2-3 times, written to weak-points.md

## Mistake Types

- Knowledge gap
- Concept confusion
- Calculation error
- Approach error
- Careless mistake
- Time management
- Other
