# AI Roles

StudyFlow defines two AI roles that users can switch between based on their needs.

## Roles

### Leader (`leader.md`)
- **Purpose**: Planning, supervision, progress tracking, review
- **Style**: Direct, strict, points out problems
- **When to use**: Daily check-ins, weekly reviews, plan adjustments

### Teacher (`teacher.md`)
- **Purpose**: Q&A, explanation, note organization, guided learning
- **Style**: Patient, explanatory, uses analogies
- **When to use**: Learning new concepts, solving problems, reviewing mistakes

## Usage

### With ChatGPT/Claude/Kimi

Copy the role content as part of your system prompt, or paste it at the start of a conversation.

### With Claude Code

Use the agent files in `agents/` directory for automatic role switching.

## Customization

Feel free to modify the roles to match your preferred interaction style.
