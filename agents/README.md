# Claude Code Agents

This directory contains agent configurations for Claude Code users. These agents can be invoked with `@agent-leader` or `@agent-teacher`.

## Setup

1. Copy the `.md` files to your project's `.claude/agents/` directory
2. Restart your Claude Code session
3. Use `@agent-leader` or `@agent-teacher` to invoke

## Agents

### Leader Agent
- Invoked with: `@agent-leader`
- Purpose: Planning, supervision, progress tracking

### Teacher Agent
- Invoked with: `@agent-teacher`
- Purpose: Q&A, explanation, note organization

## Customization

Modify the agent files to match your preferred interaction style.

## Note

These agents are optional. StudyFlow works with any AI tool, not just Claude Code.
