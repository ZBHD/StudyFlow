# StudyFlow

> 本项目积极参与并认可 [Linux.do](https://linux.do) 社区。

> 一个开源的 AI 学习协作系统，适配任意 AI 工具。

StudyFlow 帮助你用 AI 规划、追踪和改进学习。这是一个纯 Markdown 系统——无需安装软件，无需注册账号。只需要文件和一个你选择的 AI 工具。

## 特性

- **AI 协作**：让 AI 快速了解你的背景，提供个性化帮助
- **学习追踪**：每日日记、计划、复盘、间隔复习队列
- **知识管理**：笔记、错点追踪、实践记录
- **习惯养成**：统计、薄弱点追踪、进度可视化
- **通用设计**：适合学生、职场人、自学者，以及任何想系统化学习的人

## 为什么选择 StudyFlow？

### 你的困扰

- 学习没计划，想学就学，不想学就躺
- 学了就忘，复习全靠缘分
- AI 问啥答啥，但它根本不了解你
- 想让 AI 帮忙规划，它给你一堆"通用建议"
- 用了一堆学习 App，数据都在别人手里

### StudyFlow 的解法

**你的数据永远在你手里。** 换 AI 工具？把文件发给新 AI 就行。

| 方案 | 优点 | 缺点 |
|------|------|------|
| 学习 App | 界面好看 | 数据被锁定、换工具就丢 |
| 数据库 | 结构化 | 需要部署、维护 |
| **Markdown** | **通用、可读、可版本控制** | 需要自己管理文件 |

Markdown 是唯一同时满足以下条件的格式：
1. **人能读** —— 打开就能看
2. **AI 能读** —— 所有 AI 工具都支持
3. **Git 能管** —— 版本控制、历史追踪、多人协作
4. **工具无关** —— VS Code、Obsidian、记事本都能编辑

### 方案对比

| 特性 | StudyFlow | 学习 App | 纯手动 |
|------|-----------|----------|--------|
| 数据所有权 | ✅ 你的文件 | ❌ 被锁定 | ✅ 你的 |
| AI 适配 | ✅ 任意 AI | ❌ 绑定 | ❌ 无 AI |
| 学习曲线 | 低 | 低 | 高 |
| 协作能力 | ✅ Git | 有限 | ❌ |
| 价格 | 免费 | 付费 | 免费 |
| 个性化 | ✅ 完全自定义 | 有限 | ✅ |

## 快速开始

### 1. 获取系统

```bash
git clone https://github.com/yourusername/studyflow.git
```

或者下载 ZIP 并解压。

### 2. 选择你的 AI 工具

StudyFlow 适配任意 AI 工具：
- ChatGPT Codex
- Claude Code
- 任意其他 AI 工具

### 3. 初始化你的画像

打开你的 AI 工具，粘贴这段提示词：

```
我想使用 StudyFlow 学习系统。请读取 studyflow/collect/prompts/ 目录下的文件，引导我完成信息收集流程。从 basic.md 开始。
```

AI 会问你问题，并自动创建你的画像文件。

### 4. 开始学习

画像设置好后，使用这些提示词：

**制定计划：**
```
读取 core/user/profile.md 中的画像，帮我制定本周计划。
```

**学习帮助：**
```
我在学 [主题]。读取 core/user/ability-map.md 中的能力地图，从基础给我讲解 [概念]。
```

**每日汇报：**
```
让我汇报今天的进度。读取 workflow/planning/current-stage.md 中的当前计划。
```

**Claude Code 用户：**

StudyFlow 内置两个 agent，复制 `agents/` 目录到项目的 `.claude/agents/` 即可使用：
- `@agent-leader` — 规划、监督、复盘、进度追踪
- `@agent-teacher` — 答疑、讲解、笔记整理

## 架构

StudyFlow 使用分层架构：

| 层级 | 必选性 | 组件 | 用途 |
|------|--------|------|------|
| **场景包** | 可选 | 考研备考 / 技能提升 / 语言学习 | 特定场景的预设模板 |
| **内容层** | 可选 | 笔记 / 错点 / 实践记录 | 知识管理 |
| **流程层** | 核心 | 计划 / 日记 / 复盘 / 复习队列 | 学习流程管理 |
| **基础层** | 必选 | 用户画像 / AI 协作规则 / 信息收集 | AI 协作基础 |

- **基础层（必选）**：用户画像、AI 协作规则、信息收集
- **流程层（核心）**：计划、日记、复盘、间隔复习队列
- **内容层（可选）**：笔记、错点追踪、实践记录
- **场景包（可选）**：针对特定场景的预设模板

## 为什么开源？

学习不应该是私有的。

你的学习方法、你的笔记、你的复习计划——这些都是你的知识资产，不应该被任何平台锁定。

StudyFlow 用最开放的格式（Markdown）、最开放的方式（Git）、最开放的协议（MIT），让你完全掌控自己的学习数据。

**如果你也认同这个理念，欢迎 Star、Fork、PR。**
