# Scene Packages

Scene packages are pre-built templates for specific learning scenarios. They help you set up StudyFlow quickly for your particular needs.

## Available Scenes

### [Exam Preparation](exam-prep/README.md)
For students preparing for exams (college entrance, graduate school, professional certifications, etc.)

### [Skill Building](skill-building/README.md)
For people learning new skills (programming, design, writing, etc.)

### [Language Learning](language-learning/README.md)
For language learners (English, Japanese, etc.)

## Usage

1. Choose a scene that matches your situation
2. Follow the scene's setup guide
3. Customize as needed

## Creating Your Own

You can create your own scene package:

1. Create a new directory under `scenes/`
2. Include a README.md with:
   - Who this scene is for
   - Recommended module combination
   - Subject/skill list template
   - Phase planning template
   - Review strategy
   - Common weak points
