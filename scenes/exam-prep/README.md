- Students preparing for exams (college entrance, graduate school, professional certifications)
- Anyone studying for a test with a fixed date

- **Workflow (Core)**: All modules (planning, diary, review)

## Phase Planning Template

1. **Foundation Phase** (Month X - Month X)
   - Focus: Build basic understanding
   - Goals: Complete textbook chapters, understand core concepts

2. **Strengthening Phase** (Month X - Month X)
   - Focus: Deepen understanding, practice problems
   - Goals: Complete practice sets, identify weak areas

3. **Sprint Phase** (Month X - Month X)
   - Focus: Review, mock exams, fill gaps
   - Goals: Complete past papers, master weak points

## Review Strategy

- **Theory/Concepts**: Spaced repetition (D+1, D+3, D+7, D+14)
- **Problem-solving**: Regular practice + mistake review
- **Formula/Memorization**: Daily review + weekly test

## Common Weak Points

- Foundation too weak in certain subjects
- Can't remember many concepts/formulas
- Don't know how to approach certain problem types
- Time management during exams
- Anxiety and stress management
