## Learning Path Template

1. **Foundation Phase**
   - Learn basic concepts and terminology
   - Follow tutorials and examples
   - Build small practice projects

2. **Intermediate Phase**
   - Deepen understanding
   - Tackle more complex projects
   - Learn best practices

3. **Advanced Phase**
   - Real-world projects
   - Teach others or contribute to community
   - Continuous improvement

## Practice Records

Use `content/practice/templates/writeup.md` to document:
- Projects completed
- Problems solved
- Skills demonstrated
- Lessons learned

## Common Weak Points

- Understand videos but can't code/write independently
- Learn but forget quickly
- Don't know how to verify learning progress
- Stuck between tutorial hell and real projects
